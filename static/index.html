Cool Stuff

Navigating on a line

  • Beginning: ctrl-a
  • End: ctrl-e
  • Back one char: ctrl-b
  • Forward one char: ctrl-f
  • Back one word: meta-b

  • Forward one word: meta-f

  • Remap capslock to meta!

Navigating between lines

  • Previous line: ctrl-p
  • Next line: ctrl-n

Commands

  • Recursive search: ctrl-r
  • last command: !!
  • Nth last command: !-N
  • prev command: !xyz (or !?xyz?)

Recursive search is cool

$ first command
$ second command
$ <ctrl-r>fir
$ (reverse-i-search)`fir': first command

$ grep foo *.txt
$ echo bar
bar

$ cat /etc/passwd
Permission denied
$ sudo !!
sudo cat /etc/passwd
root:*:0:0:root:/var/root:/bin/sh

$ !-3
echo bar
bar

$ !?foo?
grep foo *.txt

$ !ech
echo bar
bar

Arguments

  • all arguments: !*
  • last argument: meta-. (or !!:$)
  • 3rd and fourth args: !!:3-4
  • search/replace: ^foo^bar
$ echo foo bar baz

$ echo !*
echo foo bar baz

$ echo !!:2-3
echo bar baz

$ echo !!:$
echo baz

$ ^baz^quux
echo quux

Directories

  • prev dir: cd -
  • pushd & popd
$ pwd
/etc/log/blah/blah/blah

$ pushd .

$ cd /other/directory/far/far/away
$ do my thing

$ popd
$ pwd
/etc/log/blah/blah/blah

Ultra Cool Stuff

building a complex command?

  • use ctrl-x e

need a better tail command?

  • less, then shift-f

need a quick timer?

$ time cat
ctrl-d
real  3m3.224s
user  0m0.001s
sys 0m0.003s

annoying grep output in your ps listing?

$ ps aux | grep ruby
gar 1234 /usr/bin/ruby blah.rb
gar 1235 grep ruby <--- uck!

$ ps aux | grep [r]uby
gar 1234 /usr/bin/ruby blah.rb

need to interact with a server?

$ cat < /dev/tcp/time.nist.gov/13
55581 11-01-20 23:50:52 00 0 0 812.6 UTC(NIST) *

$ echo "that's all folks!"
